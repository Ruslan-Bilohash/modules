Українська версія

Загальний огляд

Модуль Панелі Управління Адміністраторами — це адміністративний інструмент для керування обліковими записами адміністраторів. Він дозволяє додавати, редагувати та видаляти адміністраторів. Використовується PHP 8.1+, MySQL з PDO, хешування паролів (PASSWORD_BCRYPT), перевірка доступу, форми з модальними вікнами, адаптивний дизайн з Bootstrap-стилем. Модуль забезпечує безпеку через екранування вхідних даних та підготовлені запити. Інтерфейс покращено іконками Font Awesome, градієнтами та адаптивністю для мобільних пристроїв.

Список файлів та їх призначення

  • admins.php — Основний файл модуля: логіка перевірки доступу, обробка форм (додавання, редагування, видалення), вивід списку адміністраторів, HTML-інтерфейс з формами та модальними вікнами.
  • db.php — Підключення до бази даних (використовується для $conn).
  • functions.php — Функції, включаючи isAdmin() для перевірки доступу.

Основний функціонал

  • Перевірка доступу: Якщо не адміністратор, перенаправлення на логін.
  • Додавання адміністратора: Форма з логіном та паролем, хешування пароля, вставка в таблицю admins.
  • Редагування адміністратора: Модальне вікно для зміни логіна та (опціонально) пароля.
  • Видалення адміністратора: Через GET-параметр з підтвердженням.
  • Список адміністраторів: Таблиця з ID, логіном, датою створення та діями (редагувати/видалити).
  • Вивід повідомлень про успіх/помилку з іконками.
  • Отримання імені поточного адміністратора (за ID=1, з fallback на 'Фади').
  • Адаптивний дизайн: Стилі для мобільних пристроїв, градієнти, іконки Font Awesome, покращена UX з input-group.
  • Безпека: Підготовлені запити, екранування, хешування паролів, підтвердження видалення.

English Version

General Overview

The Admin Management Panel module is an administrative tool for managing admin accounts. It allows adding, editing, and deleting administrators. Uses PHP 8.1+, MySQL with PDO, password hashing (PASSWORD_BCRYPT), access check, forms with modals, responsive design with Bootstrap-style. The module ensures security through input escaping and prepared statements. Interface improved with Font Awesome icons, gradients, and mobile adaptability.

List of Files and Their Purpose

  • admins.php — Main module file: access check logic, form handling (add, edit, delete), output of admins list, HTML interface with forms and modals.
  • db.php — Database connection (used for $conn).
  • functions.php — Functions, including isAdmin() for access check.

Main Functionality

  • Access check: If not admin, redirect to login.
  • Adding admin: Form with username and password, password hashing, insert into admins table.
  • Editing admin: Modal for changing username and (optionally) password.
  • Deleting admin: Via GET parameter with confirmation.
  • List of admins: Table with ID, username, creation date, and actions (edit/delete).
  • Display success/error messages with icons.
  • Fetch current admin name (by ID=1, with fallback to 'Fadi').
  • Responsive design: Styles for mobile, gradients, Font Awesome icons, enhanced UX with input-group.
  • Security: Prepared statements, escaping, password hashing, delete confirmation.

Norsk Versjon

Generell Oversikt

Admin Management Panel-modulen er et administrativt verktøy for å administrere admin-kontoer. Den tillater å legge til, redigere og slette administratorer. Bruker PHP 8.1+, MySQL med PDO, passordhashing (PASSWORD_BCRYPT), tilgangskontroll, skjemaer med modale vinduer, responsiv design med Bootstrap-stil. Modulen sikrer sikkerhet gjennom input-escaping og forberedte spørringer. Grensesnittet er forbedret med Font Awesome-ikoner, gradienter og mobiltilpasning.

Liste over Filer og Deres Formål

  • admins.php — Hovedmodulfil: logikk for tilgangskontroll, håndtering av skjemaer (legg til, rediger, slett), utdata av admin-liste, HTML-grensesnitt med skjemaer og modale vinduer.
  • db.php — Databaseforbindelse (brukes for $conn).
  • functions.php — Funksjoner, inkludert isAdmin() for tilgangskontroll.

Hovedfunksjonalitet

  • Tilgangskontroll: Hvis ikke admin, omdiriger til innlogging.
  • Legge til admin: Skjema med brukernavn og passord, passordhashing, sett inn i admins-tabell.
  • Redigere admin: Modalt vindu for å endre brukernavn og (valgfritt) passord.
  • Slette admin: Via GET-parameter med bekreftelse.
  • Liste over admins: Tabell med ID, brukernavn, opprettelsesdato og handlinger (rediger/slett).
  • Vis suksess/feilmeldinger med ikoner.
  • Hent gjeldende admin-navn (etter ID=1, med fallback til 'Fadi').
  • Responsiv design: Stiler for mobil, gradienter, Font Awesome-ikoner, forbedret UX med input-group.
  • Sikkerhet: Forberedte spørringer, escaping, passordhashing, slettbekreftelse.
